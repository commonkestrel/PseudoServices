Google Image Search (GIS)

GoogleImageScraper

The very first of its' kind, GoogleImageScraper was programmed in Python and is by far the worst version. I had just learned about classes while making it, and decided to go with a very strange structure that is pretty inefficient in memory, time, and power. Nevertheless, it does have some very useful features, such as:

  • Various search and download arguments to refine your search
  • Fetching the standalone URLs of images without downloading
  • Fetching the source and thumbnail if needed in an object
  • Downloading images to a specific directory
  • Descriptive error names

imagesearch

imagesearch is the second iteration, built in Go. Since I comitted to doing this project in every new language, I knew I had to make the program much more streamlined, and I definitly succeeded. While it isn't perfect, this version is much more refined than GoogleImageScraper, and contains many improvements, including:

  • Cleaner and more readable source code
  • Better documentation and comments
  • Better error handling
  • Constants instead of a hashmap for options
  • Better name for ease of use

image_search

image_search is the most recent version of Google Image Search, and is the most advanced version yet. Some major improvements over the other two include:

  • Async by default, with an optional blocking API thanks to Rust's features
  • Concurrent image downloading to drastically reduce the time spent waiting for http requests
  • Enums and function chaining for options for the greatest ease of use so far
  • Retrieves width and height of image as well
  • Supports downloading thumbnails instead of full-size images.
  • More extensive options
  • Better error handling
Version: 1
Python

Google Image Search is a collection of packages in Python, Go, and Rust that I've been working on for some time. I started this project when I was making a Discord bot in Python, and needed a way to search Google Images. The only way to do this at the time was google-images-download by hardikvasa, and maintained by joeclinton1

This was an amazing and impressive project, but it had no way of just retrieving the URLs of the images, which is what I needed. Unable to find a good way of doing this, I set out on a quest to create something that had this exact functionality, which I believe I achieved. After that, it's been a goal of mine to create a way to search Google Images in every new programming language I learn.
